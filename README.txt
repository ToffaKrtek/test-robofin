Запуск:

-	При использовании Docker-compose, команда 'docker-compose up' (зарезервированный порт 8080 => http://localhost:8080)

-	Для развертывания на хостинге, прописать в  хост, порт, имя пользователя БД

Создание Базы Данных:

-	При использовании Docker-compose, происходит самостоятельно (все обращения прописаны именно под данное использование)
Под скрипты используются предложенный в задании файл create tables.sql (с дописанным созданием БД и use), а также скрипт выполняющий LOAD DATA из предложенных csv файлов -- /mysql/docker-entrypoint-initdb.d (1-init.sql и 2-pull-data.sql)

-	Для развертывания на хостинге, реализована альтернатива - php-скрипт (dbcreate.php), необходимо (после выполнения необходимых действий из инструкции по Запуску) открыть страницу {domen_name}/dbcreate.php
